# StudyBoost

Android productivity app for students.

## Features
- Task manager
- Motivation quotes
- Settings page
- Multiple screens
- ListView
- Buttons and switches

## Built With
- Java
- XML
- Android Studio


## How to Run

1. Open in Android Studio
2. Let Gradle sync
3. Run on emulator or Android device
